# Vue 3 + Vite

這個範本應該可以幫助您開始在 Vite 中使用 Vue 3 進行開發。此範本使用 Vue 3 `<script setup>` SFC，請查看[script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) 來學習更多的。
